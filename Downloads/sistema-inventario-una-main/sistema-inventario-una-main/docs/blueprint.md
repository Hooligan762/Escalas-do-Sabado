# **App Name**: Campus Inventory System

## Core Features:

- User Authentication: Secure login system with campus-specific access control.
- Inventory Management: Add, edit, and delete inventory items with details such as category, serial number, and location.
- Status Tracking: Monitor the status of each item (functioning, defective, maintenance, backup, discarded).
- Audit Logging: Track all changes to inventory items, including user, timestamp, and action.
- Reporting: Ability to export inventory data in CSV format.
- Intelligent Anomaly Detection: AI-powered tool to automatically identify potentially incorrect or missing equipment status data and report it to IT coordinators for review.
- Dashboard: Generate a dynamic overview of equipment status by category, with filter to group statistics for the various 'campus' types

## Style Guidelines:

- Primary color: Deep blue (#34495e) to convey professionalism and trust.
- Background color: Light gray (#f0f3f5) for a clean, modern look.
- Accent color: Teal (#2ecc71) to highlight key actions and information. Avoid excessive teal.
- Body font: 'PT Sans', sans-serif, for clear and accessible content.
- Headline font: 'Space Grotesk', sans-serif, for headlines and titles; provides a contemporary, tech-forward feel. Pair with 'PT Sans' for body text.
- Use clear and consistent icons to represent inventory categories and actions.
- Implement a clean, intuitive layout with clear hierarchy and easy navigation.
